# Scripts de Test de Charge

Scripts pour effectuer des tests de charge sur une API REST.

## Utilisation

### Bash (Linux/macOS)
```bash
./scripts/loadtest.sh [book_id] [requests]
```

### PowerShell (Windows)
```powershell
.\scripts\loadtest.ps1 -BookId 1 -Requests 50
```

## Paramètres

- `book_id` / `BookId` : Identifiant de la ressource (défaut: 1)
- `requests` / `Requests` : Nombre de requêtes (défaut: 50)
